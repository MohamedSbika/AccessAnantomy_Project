Préparations nécessaires pour travailler avec la MERN Stack

Node.js + NPM

Rôle : Permet d’exécuter JavaScript côté serveur et d’installer les dépendances du projet.

Lien de téléchargement : https://nodejs.org

MongoDB Community Edition

Rôle : Base de données NoSQL utilisée pour stocker les données.

Lien de téléchargement : https://www.mongodb.com/try/download/community

MongoDB Compass

Rôle : Interface graphique pour visualiser les bases de données MongoDB.

Lien de téléchargement : https://www.mongodb.com/try/download/compass

Visual Studio Code (VS Code)

Rôle : Éditeur de code recommandé pour les projets MERN.

Lien de téléchargement : https://code.visualstudio.com/

Postman

Rôle : Permet de tester les API backend via des requêtes HTTP.

Lien de téléchargement : https://www.postman.com/downloads/

Ps : Activer l’option add to path si demandé

Vérifier Node & NPM

node -v

npm -v

Vérifier MongoDB

mongod --version

Si MongoDB s'exécute :

mongod

Puis dans un autre terminal :

mongo
